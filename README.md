# Bric-a-Break

A browser game clone of breakout.

### MVP

A game where the user controls a paddle preventing a ball from reaching the bottom of the screen and bouncing it to break bricks.

* [ ] Paddle and player control (MVP)
* [ ] Ball (MVP)
  - [ ] bounces off paddle

* [ ] Bricks (MVP)
